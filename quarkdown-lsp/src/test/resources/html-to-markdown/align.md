```block lang-kotlin
.align alignment:{Container.Alignment} body:{MarkdownContent} -> Node
```

Aligns content and text within its parent.

#### Return

the new aligned Container node

#### Parameters

* **alignment**


  content alignment anchor and text alignment

  Values
  * `start`
  * `center`
  * `end`

* **body**


  content to center
